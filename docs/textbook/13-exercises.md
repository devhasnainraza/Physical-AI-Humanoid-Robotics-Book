# Exercises

## Set 1: Kinematics
1.  Derive the DH Parameters for a 3-DOF planar arm.
2.  Calculate the Jacobian $J(q)$ for the same arm.
3.  Find the singular configurations.

## Set 2: Control
1.  Tune a PID controller for a mass-spring system $m=1, k=10$.
2.  Implement an Impedance Controller in Python simulation.

## Set 3: RL
1.  Prove the Bellman Optimality Equation.
2.  Implement Q-Learning for GridWorld from scratch (numpy only).
