# Lab: Creating a URDF for a Humanoid Robot

In this lab, you will learn how to create a Unified Robot Description Format (URDF) file for a humanoid robot.

## Objectives
- Understand the structure of a URDF file.
- Define links and joints.
- Visualize your robot in RViz.

## Steps
1. Define the base link (torso).
2. Add legs with hip, knee, and ankle joints.
3. Add arms with shoulder and elbow joints.
4. Add the head.
